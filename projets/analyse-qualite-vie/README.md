# 🌍 Analyse de la qualité de vie par pays

## Objectif
Analyser différents indicateurs socio-économiques afin de comparer la qualité de vie entre plusieurs pays.

## Outils
- Python
- Pandas
- Matplotlib

## Résultats
- comparaison d’indicateurs par pays
- mise en évidence des écarts de qualité de vie
- visualisations claires et interprétables

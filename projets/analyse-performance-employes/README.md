# 👥 Analyse des performances des employés

## Objectif
Analyser la distribution des performances et des heures travaillées afin d’identifier les écarts et les outliers.

## Outils
- Python
- Pandas
- Matplotlib

## Compétences
- Statistiques descriptives
- Visualisation des distributions
- Détection d’outliers (IQR)
- Analyse de données RH

## Résultats
- Identification des employés atypiques
- Analyse des distributions
- Mise en évidence des écarts de performance

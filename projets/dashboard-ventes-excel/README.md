# 📊 Dashboard des ventes (Excel / LibreOffice)

## 🎯 Objectif
Analyser les performances des ventes par région et catégorie à l’aide d’un dashboard interactif.

## 🧰 Outils utilisés
- Excel / LibreOffice Calc
- Tableaux croisés dynamiques
- KPI dynamiques
- Graphiques interactifs

## 📊 Fonctionnalités
- Filtre par région et catégorie
- KPI : ventes, profit, marge
- Graphiques dynamiques
- Dashboard interactif

## 📸 Aperçu
![Dashboard](dashboard.png)

## 💡 Résultats
- Identification des régions performantes
- Analyse des catégories les plus rentables
- Visualisation claire des tendances

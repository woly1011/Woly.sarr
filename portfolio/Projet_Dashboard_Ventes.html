Projet Data – Dashboard des ventes

Excel · LibreOffice Calc · Python · KPI · Data Visualization

Résumé du projet

Ce projet a pour objectif de suivre et analyser les performances commerciales à partir du dataset Superstore. Il permet d’examiner les ventes, les profits et les marges selon les régions et les catégories de produits afin d’identifier les zones les plus performantes et celles à améliorer.

Le projet a abouti à la création d’un dashboard interactif facilitant la lecture des indicateurs clés et la prise de décision. L’analyse a également été documentée dans un notebook explicatif afin de présenter les étapes de préparation, de calcul et d’interprétation des résultats.

Pipeline d’analyse

1. Importation du dataset Superstore
2. Nettoyage et vérification des données
3. Structuration des variables utiles à l’analyse
4. Calcul des KPI : ventes, profit, marge
5. Analyse par région et par catégorie
6. Création de tableaux croisés dynamiques
7. Mise en place de filtres interactifs
8. Création du dashboard dans Excel / LibreOffice Calc
9. Documentation de l’analyse dans un notebook Python
